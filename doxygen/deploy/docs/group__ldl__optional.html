#define LORA_ERROR (   ...)

Example (defined from a makefile):

1 -DLORA_ERROR(...)='do{printf(__VA_ARGS__);printf("\n");}while(0);'

This could also be defined within the file included by LORA_TARGET_INCLUDE.

#define LORA_MAX_PACKET   UINT8_MAX

Redefine this to reduce the stack and data memory footprint.

Example (defined in makefile):

1 -DLORA_MAX_PACKET=64U
#define LORA_XTAL_ERROR   0UL
  • Redefine this to account for oscillator +/- error per second in ticks
  • This is used to compensate for the fastest and slowest clocks

For example, if an oscillator is accurate to +/1% of nominal:

1 F_CPU := 8000000UL
2 PRESCALE := 64UL
3 ERROR := 0.01
4 
5 # works out to 1250 ticks
6 -DLORA_XTAL_ERROR='( $(F_CPU) * $(ERROR) / $(PRESCALE) )'

Note that LDL will multiply by two in order to account for the full range of slow vs. fast.

#define LORA_TICK_ADVANCE   0UL
  • this advances RX1 and RX2 window schedule by so many system time ticks
  • this is likely to be negligible once XTAL error is accounted for

example:

1 -DLORA_TICK_ADVANCE=7UL
#define LORA_DEFAULT_RATE   2U
  • transmit rate is set to this value upon reset
  • if ADR is enabled, the stack will never drop back below this value

example:

1 -DLORA_DEFAULT_RATE=0U

This would ensure the stack defaults to using the largest spreading factor.

#define LORA_TARGET_INCLUDE

Include a target specific file in all headers.

Note
This will ensure LORA_TARGET_INCLUDE is included in every LDL source and header file

Within this file you could define things like LORA_ERROR() and LORA_ASSERT(). Also use this file to include lora_aes_ctx and lora_cmac_ctx if they have been changed.

Example (defined from a makefile):

1 -DLORA_TARGET_INCLUDE='target_specific.h'

Where target_specific.h would be kept somewhere on the include search path.

#define LORA_ENABLE_PLATFORM_AES

Define this macro to remove the built-in AES implementation.

The effect of defining this macro is that:

  • lora_aes.c will build to an empty object
  • struct lora_aes_ctx is only a forward declaration
